포트 개념 이해

IP = 집주소
포트 = 아파트 몇 호
하나의 서버 안에 여러 서비스가 있다.
서버는 특정 포트를 열어 서비스 제공
내 PC도 통신할 때 포트를 사용한다.


반드시 기억해야 할 포트 3개
- 80 > HTTP
- -443 > HTTPS
-  22 > SSH

포트가 필요한 이유
하나의 IP 안에서 여러 서비스를 구분하기 위한 번호이다.
(포트가 없으면 어느 프로그램에게 데이터를 줘야 할지 모름.)
