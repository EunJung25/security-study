CSRF
어떨게 하면 브라우저가 POST/transfer를 자동으로 보내게 할까?
-공격자가 만든 페이지에 자동 제출되는 from또는 스크립트가 있기 때문에
-CSRF는 로그인 상태의 브라우저가 공격자가 만든 요청을 자동으로 보내게 만드는 공격이다.


기초 중에 헷갈린 부분 복습
GET/POST차이
GET
-서버에 데이터를 가져올때
POST
-서버에 데이터를 보낼 때


CSRF는 브라우저로 요청 강제 전송
XSS는 브라우저에서 스크립트 실행

둘의 공통점은 브라우저가 무대인 점
다른 점은 공격 방식이 다르다
